The MINTverse Ecosystem

🎯 MINTer

The main R package that orchestrates the entire modeling pipeline. Provides the simple run_mint_scenarios() function for end-to-end analysis.

  • One-function workflow
  • Handles bed net combinations
  • Projects future interventions
  • Automatic visualization

📈 estiMINT

Converts field prevalence data into EIR estimates and predicts annual case incidence using machine learning.

  • XGBoost & Random Forest models
  • Prevalence → EIR conversion
  • Case incidence predictions
  • Pre-trained models included

🧠 MINTe

Neural network time series forecasting using LSTM and GRU architectures for prevalence predictions.

  • LSTM & GRU networks
  • GPU acceleration
  • Automatic hyperparameter tuning
  • Time series specialists

Quick Start

Get predictions in seconds with just one function call 

# Install MINTer (includes all dependencies)
devtools::install_github("CosmoNaught/MINTer")

# Load the package
library(MINTer)

# Run comprehensive scenarios with one function
results <- MINTer::run_mint_scenarios(
  # Net combination settings
  res_use        = c(0.30, 0.45, 0.20),   # Current resistance
  res_future     = c(0.60, 0.75, 0.50),   # Future resistance
  py_only        = c(0.40, 0.30, 0.50),   # Pyrethroid-only nets
  py_pbo         = c(0.10, 0.15, 0.05),   # Pyrethroid-PBO nets
  py_pyrrole     = c(0.00, 0.05, 0.00),   # Pyrethroid-pyrrole
  py_ppf         = c(0.10, 0.10, 0.15),   # Pyrethroid-PPF
  
  # Malaria environment settings
  prev_vec       = c(0.20, 0.40, 0.60),   # Current prevalence
  Q0_vec         = c(0.65, 0.75, 0.85),   # Human blood index
  phi_vec        = c(0.45, 0.65, 0.75),   # Bites in bed
  season_vec     = c(0, 1, 1),            # Seasonality
  routine_vec    = c(0, 0, 1),            # Routine distribution
  irs_vec        = c(0.10, 0.35, 0.70),   # Current IRS coverage
  irs_future_vec = c(0.20, 0.40, 0.50),   # Future IRS coverage
  lsm_vec        = c(0.05, 0.45, 0.85)    # LSM coverage
)

# Save results
write.csv(results$prevalence, "results_prevalence.csv", row.names = FALSE)
write.csv(results$cases, "results_cases.csv", row.names = FALSE)

# Create beautiful visualizations
create_scenario_plots(results$prevalence, output_dir = "plots")

Key Parameters

Parameter Description Range Example
res_use Current pyrethroid resistance level 0-1 0.30 (30% resistance)
res_future Projected resistance after intervention 0-1 0.60 (60% resistance)
py_only Proportion of pyrethroid-only nets 0-1 0.40 (40% of nets)
prev_vec Current malaria prevalence (PfPR2-10) 0-1 0.20 (20% prevalence)
Q0_vec Human blood index (anthropophagy) 0.5-0.9 0.65
phi_vec Proportion of bites occurring in bed 0-1 0.45
irs_vec Current IRS household coverage 0-1 0.10 (10% coverage)
lsm_vec Larval source management coverage 0-1 0.05 (5% coverage)
